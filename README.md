## Aplikasi SmartQuiz
SmartQuiz adalah aplikasi kuis edukatif berbasis desktop yang dirancang untuk membantu meningkatkan minat belajar anak melalui kuis interaktif dengan pendampingan. Aplikasi ini menyediakan mode siswa untuk mengerjakan kuis dan mode guru (pendamping) untuk membuat serta mengelola soal.
## Fitur Utama
- Mode Siswa untuk mengerjakan kuis
- Mode Guru (Pendamping) untuk membuat dan mengelola soal
- Pilihan kategori kuis:
  - Literasi Dasar
  - Bahasa Indonesia
  - Pengetahuan Umum
  - Matematika
- Bank soal default per kategori
- Soal pilihan ganda dengan opsi gambar
- Penilaian otomatis dan tampilan skor
- Penyimpanan riwayat hasil kuis
- Aplikasi bersifat single user
## Cara Menjalankan Aplikasi (Source Code)
1. Unduh source code aplikasi SmartQuiz.
2. Ekstrak file project ke dalam satu folder.
3. Pastikan library PySide6 sudah terinstal.
4. Buka aplikasi Visual Studio Code
5. Pilih menu File â†’ Open Folder, lalu pilih folder project SmartQuiz yang telah diekstrak.
6. Pastikan seluruh file dan folder pendukung (folder ui, images, dan file database) berada dalam folder project yang sama.
7. Buka file main.py.
8. Jalankan aplikasi dengan menekan tombol Run pada Visual Studio Code.
## Cara Menggunakan Aplikasi
- Mode Siswa: Memilih kategori kuis dan mengerjakan soal.
- Mode Guru: Membuat, melihat, dan menghapus soal kuis.
- Riwayat: Melihat dan menghapus hasil kuis yang telah dikerjakan.
## Catatan
